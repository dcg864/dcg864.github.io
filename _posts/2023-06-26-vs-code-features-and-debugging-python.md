s4int1zzy has been developing python code for around 4 years and his text editor of choice is VS Code. One of his struggles during his formative years writing scripts was improving his workflow so he could get working code deployed as soon as possible. Thus, he dedicated time trying to understand all the basic features of VS Code as well as some advanced techniques. In this talk, s4int1zzy will walk through his lessons learned and guide you through some basic, yet lesser known, features of VS Code and some techniques to help you debug and refactor your python code.
